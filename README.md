# NewsCore

منصّة خلفية حديثة وعالية الأداء لبناء مواقع إخبارية ذكية. تضم إدارة المحتوى، جلب الأخبار، التحسين بالذكاء الاصطناعي، التحليلات، والدعم متعدد اللغات، مبنية على هندسة قابلة للتوسع.

## البدء السريع

1. انسخ ملف البيئة الافتراضي:

   ```bash
   cp .env.example .env
   ```

2. حدّث متغيرات البيئة الضرورية مثل `DATABASE_URL` و`JWT_SECRET`.

3. ثبّت الاعتماديات وشغّل الخادم في وضع التطوير:

   ```bash
   npm install
   npm run start:dev
   ```

> ملاحظة: في بعض البيئات المقيّدة قد تحتاج إلى ضبط إعدادات الـ proxy للوصول إلى سجل npm.

## بنية المشروع

- `src/main.ts`: تهيئة تطبيق NestJS مع CORS، Helmet، الضغط، ونسخة API.
- `src/app.module.ts`: تحميل الإعدادات العامة والتحقق من متغيرات البيئة.
- `src/config/`: ملفات إعدادات التطبيق والتحقق من القيم.
- `src/health/`: نقطة فحص بسيطة للتأكّد من جاهزية الخدمة.
- `src/database/prisma.service.ts`: خدمة Prisma موحدة للاتصال بقاعدة البيانات وإدارة الإغلاق السليم.

## متغيرات البيئة الأساسية

اطّلع على `.env.example` لمجموعة المتغيرات المدعومة، ومن أهمها:

- `DATABASE_URL`: اتصال PostgreSQL.
- `JWT_SECRET`: مفتاح التوقيع.
- `S3_ENDPOINT`, `S3_BUCKET`: إعدادات التخزين.
- `THROTTLE_TTL`, `THROTTLE_LIMIT`: التحكم في معدل الطلبات.
